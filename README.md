## Репозиторий для итоговой проверочной работы 

### Задача:
1. Создать репозиторий на GitHUB
2. Нарисовать блок-схему алгоритма или метода
3. Снабдить репозиторий текстовым описанием решения
4. Написать программу, которая из имеющегося массива строк формирует массив из строк, длина которого меньше или равна трем символам. Первоначальный массив можно ввести с клавиатуры или задать на старте выполнения.
5. Использовать контроль версий в работе над этим проектом.

### Решение:
Объявляется два массива одинаковой длины. Создается метод 
FormSecondArrayFrom, в котором с помощью цикла for перебираются все элементы первого массива и при соответствии условию(<=3 символов строки) заносятся в count элемент второго массива.
Сформированный массив выводится с помощью метода PrintArray.
Блок-схема метода FormSecondArrayFrom приложена к задаче в файле.
